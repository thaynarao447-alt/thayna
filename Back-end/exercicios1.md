**1.** O que é o Node.js?
é um ambiente de tempo de execução (runtime) de código aberto e multiplataforma que permite executar JavaScript no lado do servidor (backend), fora de um navegador web

**2.** O Node.js usa qual linguagem de programação?
JavaScript. 

**3.**O Node.js roda no navegador ou fora dele?
O Node.js roda fora do navegador

**4.** Para que serve o Node.js?
principalmente para executar código JavaScript no lado do servidor (back-end), permitindo o desenvolvimento de aplicações web rápidas, escaláveis e em tempo real

**5.** Qual comando é usado para executar um arquivo Node.js no terminal?
O comando principal para executar um arquivo Node.js no terminal é node <nome_do_arquivo.js>.

**6.** O que é o **npm**?
é o gerenciador de pacotes padrão para o ambiente de execução JavaScript Node.js

**7.**O que significa a palavra módulo no Node.js?
é o gerenciador de pacotes padrão para o ambiente de execução JavaScript Node.js

**8.** O Node.js é mais usado no **front-end** ou no **back-end**?
O Node.js é esmagadoramente mais usado no back-end

**9.** O que é o arquivo **package.json**?
O arquivo package.json é o coração de qualquer projeto Node.js ou de desenvolvimento frontend moderno (React, Vue, Angular, etc.)

**10.** O Node.js é gratuito ou pago?
gratuito
